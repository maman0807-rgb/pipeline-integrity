Pipeline Integrity System — PHR Prabumulih Field

🛢️ Tentang Sistem

Pipeline Integrity System adalah aplikasi web berbasis PWA untuk memantau, mengelola, dan menganalisis kondisi integritas pipa di Prabumulih Field secara real-time.

Sistem ini menggantikan pencatatan manual di Excel dan memberikan visibilitas terpusat terhadap seluruh aset pipa — dari data register, hasil inspeksi, riwayat kebocoran, hingga analisis kelayakan penggantian.

Masalah Lama
Data tersebar di banyak file Excel, tidak real-time, sulit dianalisis
Solusi Sistem
Satu platform terpusat, update otomatis, analisis risiko visual

Cara menggunakan

  1. Pantau card Status BAD dan P1 — jika angka tinggi, ada pipa yang butuh perhatian segera
  2. Cek tabel "COI/PLO Akan Berakhir" — perpanjang sertifikat sebelum lewat tanggal
  3. Lihat 5 kejadian bocor terbaru di panel kanan bawah
  4. Klik Refresh di pojok kanan atas untuk update data tanpa reload halaman

🔗 2. Flowline Register

Master data seluruh flowline — pipa dari sumur ke stasiun pengumpul. Data di sini adalah register resmi aset pipa.

Data yang dikelola

FieldKeterangan
Dari Sumur → Ke StasiunIdentitas titik awal dan akhir pipa
Panjang (m)Panjang total flowline
Tahun KonstruksiTahun pipa dipasang
PriorityP1 Kritis  P2 P3 P4 Normal
Integrity StatusGOOD  MONITOR  BAD
Sertifikat COI/PLONomor dan tanggal berlaku sertifikat

Cara menggunakan

  1. Klik + Tambah untuk input flowline baru
  2. Klik ikon ✏️ pensil untuk edit data yang sudah ada
  3. Ketik di kotak Cari untuk filter berdasarkan nama sumur, cluster, atau status
  4. Klik Export untuk download semua data ke file Excel (.xlsx)

🔬 3. Monitoring Inspeksi

Data teknis hasil inspeksi per segmen pipa. Lebih detail dari Flowline Register karena mencatat kondisi teknis aktual.

Data yang dikelola

FieldKeterangan
KategoriFLOWLINE / TRUNKLINE / INJECTION LINE
Diameter (inch)Ukuran nominal pipa
Corrosion RateLaju korosi dalam mm/tahun
Remain LifeEstimasi sisa umur pakai (tahun)
Leak EventJumlah bocor — otomatis update dari History Kebocoran
Integrity StatusHasil akhir penilaian kondisi segmen

Cara menggunakan

  1. Pilih filter ALL / GOOD / MONITOR / BAD untuk melihat per kategori status
  2. Ketik nama lokasi di kotak pencarian untuk cari segmen spesifik
  3. Klik ✏️ pensil untuk update hasil inspeksi terbaru (corrosion rate, remain life, status)
  4. Klik Export untuk download data ke Excel
⚠️ Kolom Leak Event tidak perlu diisi manual — akan otomatis bertambah setiap kali kebocoran baru dicatat dengan segmen yang dipilih.

🔥 4. History Kebocoran

Pencatatan dan penelusuran seluruh riwayat kejadian kebocoran pipa. Total lebih dari 5.500 data historis tersimpan di sistem.

Cara mencatat kebocoran baru

  1. Klik + Catat di pojok kanan atas
  2. Pilih Segmen Pipa dari dropdown — field Lokasi akan auto-terisi sesuai segmen
  3. Isi tanggal kejadian, tanggal mulai & selesai perbaikan
  4. Isi jumlah: Bocor (titik), Clamp (titik), Sadel (titik), Sisip (m)
  5. Klik ✓ Simpan — sistem otomatis update Decision Matrix

Summary cards

Empat kartu di bagian atas menampilkan total seluruh data sesuai filter pencarian yang aktif — bukan hanya halaman yang sedang ditampilkan.

✅ Setelah menyimpan kebocoran baru, buka Decision Matrix — segmen terkait otomatis berpindah ke zona risiko yang lebih tinggi.

📐 5. Decision Matrix

Visualisasi risiko seluruh segmen pipa dalam format matriks PoF × CoF. Membantu menentukan prioritas penanganan berbasis risiko.

Cara membaca matriks

PoF — Probability of Failure
PoF 10–1× bocor
PoF 22–3× bocor
PoF 34–7× bocor
PoF 48–14× bocor
PoF 5≥15× bocor
CoF — Consequence of Failure
CoF 5Status BAD
CoF 4MONITOR + Gas
CoF 3MONITOR + Oil
CoF 2GOOD + Gas
CoF 1GOOD + Oil/Lain

Risk Level

R1 — Low R2 — Low-Med R3 — Medium R4 — High R5 — Extreme

Cara menggunakan

  1. Lihat sel matriks yang paling merah (R4-R5) — segmen di sini adalah prioritas utama
  2. Klik sel manapun untuk melihat daftar segmen di posisi risiko tersebut
  3. Panel detail muncul di bawah — tampilkan nama segmen, status, dan jumlah bocor
  4. Matrix otomatis berubah setiap ada kebocoran baru atau update status inspeksi

💰 6. CBA Kalkulator

Menghitung kelayakan ekonomi penggantian pipa (Cost-Benefit Analysis). Output berupa NPV dan payback period untuk justifikasi pengajuan budget.

Cara menggunakan

  1. Pilih Segmen Pipa dari dropdown di bagian "Auto-fill dari Database"
  2. Klik "Isi Otomatis dari Segmen Ini" — data teknis terisi otomatis:
    diameter, panjang, umur pipa, frekuensi bocor/tahun, toggle Oil/Gas
  3. Cek apakah data teknis sudah benar — bisa diedit manual jika perlu
  4. Isi Parameter Ekonomi: harga minyak/gas, biaya mobilisasi, gaji crew, WACC, dll
  5. Tentukan Panjang Ganti (m) — berapa meter pipa yang akan diganti
  6. Klik Hitung CBA

Membaca hasil

OutputArtinya
Kerugian TahunanTotal kerugian produksi + biaya darurat per tahun akibat bocor
Total CAPEXEstimasi biaya penggantian pipa (material + mobilisasi + crew + mesin)
NPVJika positif → penggantian LAYAK. Jika negatif → belum layak
PaybackBerapa tahun modal penggantian kembali dari penghematan kerugian bocor
💡 Hasil CBA bisa digunakan sebagai lampiran justifikasi ekonomi dalam pengajuan budget penggantian pipa ke manajemen.

👥 7. Manajemen User (Admin only)

Mengelola akun pengguna sistem. Hanya bisa diakses oleh role Admin.

Cara tambah user baru

  1. Isi form: Username, Nama Lengkap, NIP, Role
  2. Klik Generate SQL
  3. Copy SQL yang muncul → paste di Supabase SQL Editor → Run
  4. Berikan email dan password kepada user baru

📥 8. Import Excel (Admin only)

Upload data massal dari file Excel ke database. Berguna untuk migrasi data awal atau penambahan data dalam jumlah besar.

Format file Excel

SheetIsi
Sheet 1Data Flowline (master register)
Sheet 2Data Segmen Monitoring Inspeksi
Sheet 3Data History Kebocoran

Cara menggunakan

  1. Klik Pilih File Excel → pilih file .xlsx
  2. Sistem menampilkan preview jumlah baris per sheet yang akan diimport
  3. Sistem otomatis skip duplikat — data yang sudah ada tidak akan diimport ulang
  4. Klik Import untuk memasukkan data ke database
